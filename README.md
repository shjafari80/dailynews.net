ویژگی‌های پروژه
•	فرانت‌اند: HTML + CSS + Bootstrap 5 (ریسپانسیو کامل، تم آسمان شب زیبا)

•	دیتابیس: SQLite (فایل محلی DailyNews.db - بدون نیاز به SQL Server)
•	ORM: Entity Framework Core (Code-First + Migration)
•	پنل مدیریت (Admin Area): 
o	CRUD کامل برای اخبار (ایجاد، ویرایش، جزئیات، حذف)
o	لیست دسته‌بندی‌ها و نویسندگان
o	جدول با عملیات سریع و تأیید حذف
